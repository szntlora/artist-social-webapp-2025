MuseXion is a modern full-stack web platform built for artists to share their work, connect with others, and grow their digital presence.
It offers a complete community experience: artwork feed, comments, likes, tags, categories, profiles, and real-time notifications.

Key Features
- JWT-based authentication (registration with email code, login, password reset)
- Artwork upload system with multi-image support, categories, and tags
- Comment & Like system with dynamic modal windows
- Filtering by categories, tags, keywords, and usernames
- Artist profiles: follow/unfollow and count, profile customization
- Notification center (likes, comments, mentions)
- Global settings: email/password change, account deletion, data export
- Dark/Light theme toggle

Tech Stack:
Backend: Python, Django, Django REST Framework, MySQL, JWT
Frontend: HTML, CSS, JavaScript
Additional: Django Filters, Pillow, CORS
